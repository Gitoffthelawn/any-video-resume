# Installation

## FireFox

1. Go to FireFox Addon page and activate it
2. Thats all, now you can use the extension.

## Chrome/Edge: 

1. Go to the Chrome store page and activate it
2. Thats all, now you can use the extension.

## Manual installation

1. Download zip or xpi file
2. Upload it to your browser und activate  it.
